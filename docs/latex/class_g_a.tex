\subsubsection*{template$<$typename T$>$\newline
class GA$<$ T $>$}
Template class representing a Genetic Algorithm (\doxylink{class_g_a}{GA}). 

The \doxylink{class_g_a}{GA} class encapsulates the operations and parameters required to perform a \doxylink{class_g_a}{GA} optimization. The \doxylink{class_g_a}{GA} class is completely abstract and can be used for any optimization problem.
