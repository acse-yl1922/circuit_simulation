class Individual$<$ T $>$}
Template class representing an individual in the \doxylink{class_g_a}{GA} population. 

An \doxylink{class_individual}{Individual} consists of a chromosome (a vector of a specific type), representing its genetic information, and a fitness value indicating the quality of the individual\textquotesingle{}s solution.
